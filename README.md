# Rubidium API

Transformer mini-GPT en PyTorch con FastAPI.

## Endpoints

- `GET /` — estado del servicio
- `GET /state` — estado del modelo
- `POST /generate` — generar texto
- `POST /train` — entrenar modelo
- `POST /save` — guardar modelo
- `POST /load` — cargar modelo
